page accueil / premiere ligne :

Découvrez, apprenez et améliorez vos compétences sur des modules scientifiques innovants et variés.  
Profitez d'un temps de calcul réduit grâce au calcul distribué sur cluster de Raspberry Pi en vous inscrivant dès maintenant.

page accueil / description module 1 : 

Le premier module de notre site ! Découvrez la puissance des nombres premiers, explorez la théorie des nombres 
et optimisez vos algorithmes de la manière la plus simple qui soit.

page accueil / description module 2 :

Découvrez la puissance du célèbre algorithme de Monte Carlo dans un module permettant de calculer une valeur approchée de Pi. 
(Et amusez-vous à en retenir les premières décimales pour impressionner vos proches..)

page accueil / description module 3 : 

Expérimentez et découvrez l'IA ainsi que ses capacités hors du commun dans un module innovant de Machine Learning dédié à la reconnaissance
de voitures de plus de 20 constructeurs automobiles.
